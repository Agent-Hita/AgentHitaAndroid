Agent Hita is built on the principle: "Safety should never come at the cost of privacy." Our goal is to maximize on-device processing and minimize data collection.

Information We Collect

We may collect:

  • Device Information — device model, operating system version, application version.
  • Technical Information — crash logs, diagnostic information, performance metrics.
  • Usage Information — feature usage statistics, configuration settings, service health information.

On-Device Processing

  • AI analysis occurs on-device.
