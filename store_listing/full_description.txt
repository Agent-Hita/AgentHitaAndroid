HOW IT WORKS

Agent Hita uses Android's Accessibility Service to read message text as it appears on your screen — the same way a screen reader works. This is the only way to analyse conversations across all messaging apps, including end-to-end encrypted ones, without routing your messages through any server.

The Accessibility Service reads text from the following apps only:

- WhatsApp
- Instagram Direct
- Google Messages
- Samsung Messages
- Android SMS / MMS apps

Text is analysed instantly in memory and never written to disk or transmitted off your device. The Accessibility Service is not used for any other purpose. You can revoke access at any time in Settings → Accessibility → Agent Hita.

GUARDIAN ALERTS

Enable Guardian Alerts to notify a trusted contact if Agent Hita detects a high-risk threat pattern. Your guardian receives an alert with only the threat category and risk level — never any message content. You stay in control of who is notified and when.
